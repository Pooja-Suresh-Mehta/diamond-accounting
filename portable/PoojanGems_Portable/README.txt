===================================================
  POOJAN GEMS - Diamond Accounting (Portable)
===================================================

FIRST TIME ON A NEW COMPUTER:
  Windows:  Double-click "setup-windows.bat"
            (downloads Python automatically, ~25 MB, one-time)
            Then double-click "start.bat"

  Mac:      Python 3 is pre-installed on most Macs.
            Double-click "start.sh" or run: ./start.sh
            If not: brew install python3

  Linux:    sudo apt install python3 python3-venv python3-pip
            Then run: ./start.sh

AFTER FIRST SETUP:
  Just double-click "start.bat" (Windows) or "start.sh" (Mac/Linux)

RESOURCE USAGE:
  RAM:  ~60-100 MB (very lightweight)
  CPU:  < 1% when idle
  Disk: ~200 MB (Python + dependencies)

DATA:
  On pendrive:   data/diamond_accounting.db
  Local backup:  C:\PoojanGems_Backup  (Windows)
                 ~/PoojanGems_Backup   (Mac/Linux)
  Auto-backed up every time you start/stop the app.

BACKUP FROM INSIDE THE APP:
  Sidebar > "Backup & Restore"
  - Download as Excel (.xlsx) - human-readable backup
  - Download SQLite DB - exact database copy
  - Restore from Excel - upload a backup to restore

LOGIN:  admin / admin123
